What is the Document Object Model (DOM)

Document means HTML document, Object is a javascript entity with specific property and structure and finally DOM is the object representation of any HTML page. DOM is Tree look shape and it has many nodes

How do you select an element form the DOM?

We can select element by two main way, i.e., Single Element Selector and Multiple Element Selector. querySelector or getEelementByID is the example of Single Element Selector whereas getElementsClassName is the example of Multiple Element Selector.

What is event delegation in the context of the DOM, and why it is useful?

Event Delegation is system to the set addEventListener property to the parent element instead of child. It is useful when we have multiple child element and we don`t need the set addEventListener for everyone

How do you manipulate an element`s attributes and styles using the DOM?

To manipulate an element`s attributes we can use setAttribute() method. JavaScript also allows its users to change the style of any particular element with the help style keyword.
